## Core educational logic

Week 1 now separates observation, responsibility, decision threshold, and actual decision work.

- Baseline / map-observation work is where students view, analyze, and defend.
- The Academy sequence is not a worksheet. It prepares students emotionally and intellectually.
- The Decision Threshold page does not ask the final settlement question; it sends students to the map.
- The Strategic Map is where the actual settlement choice is made and defended.

Core distinction:

- Baseline Inquiry asks: **What do you see?**
- Decision Threshold asks: **What will you do with what you saw?**
- Strategic Map Defense requires: **Choose and defend with evidence.**

## Design locks

### Academy visual identity

- Ancient-future Academy atmosphere.
- Observatory Compass / Academy seal.
- Warm gold and deep charcoal palette.
- Haze, particles, and chamber depth.
- Page 1 remains darker and more reflective.
- Page 2 uses Academy Memory Chamber background.
- Page 3 uses a darker threshold version of the memory chamber.

### Audio behavior

- Each Academy page uses a user click/start gate to avoid browser autoplay blocking.
- Audio and text should start together.
- Audio must match visible text.
- Fadeouts should not cut off narration.

### Pedagogical rule

Do not ask terrain/settlement defense questions before a visible reference/context exists.

- Page 2 remains philosophical.
- Page 3 is a threshold.
- `/map` is where the actual defense occurs.

## Current known limitations

4. Printable archive uses browser print/save-as-PDF rather than server PDF generation.
5. Project Hub is still a functional placeholder, not yet a full dashboard.
6. The Strategic Map is still a structured decision workspace, not yet an interactive geographic simulation map.
