# STEM Nation Student Avatar Portrait Prompts

## Visual Standard

Council-style illustrated portraits for STEM Nation student avatar selection.

Use:
- animated-series / high-quality cartoon portrait style
- middle-grade student leaders
- deep navy, warm gold, earth tones, river blue, forest green, parchment accents
- expressive heroic student faces
- clean portrait-card framing
- no text in image
- no emoji
- same visual universe across all six avatars

## File Targets

Save final images as:

- `stem-nation/static/avatars/student_scout.png`
