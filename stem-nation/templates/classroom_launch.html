Classroom Launch Panel

A teacher-facing control page for starting the student simulation, guiding the room, and returning to the command center during instruction.

Recommended Classroom Sequence

  1. Open the Student Map and ask students to observe terrain first.
  2. Have students select a settlement region and defend their reasoning.
  3. Return to the Command Center to view saved civilization evidence.
  4. Use Demo Cleanup Options only when preparing for a fresh demonstration.
  5. Use Student Mode Reset only when beginning a completely new browser session.
